zip contents
------------
tournament.py		Tournament API
tournament.sql		SQL Script to create necessary TABLEs and VIEWs
tournament_test.py	Program to test API



Installation
------------
1. Unzip tournament.zip into your chosen installation directory
2. Load and execute tournament.sql in your chosen POSTGRES database to create necessary TABLEs and VIEWs


To Test API
-----------
1. Execute tournament_test.py to test tournament.py functionality
